After the current Step 1 block (which ends at the `---` on line 66), insert a new step before Step 2:

```markdown
## 1.5. Roadmap Sync Check

<HARD-GATE>
Before pausing, verify that ROADMAP.md reflects the actual state of completed work. This is a safety net for cases where `/execute` Step 6 was skipped.
</HARD-GATE>

**Check for unsynchronized phase completions:**

1. Read `.quantis/ROADMAP.md` and identify all phases that are NOT marked `✅ Complete`
2. For each such phase, check if a phase directory exists with SUMMARY.md files:
   ```bash
   # Find phase directories with completed summaries
   for dir in .quantis/phases/*/; do
       if ls "$dir"*-SUMMARY.md 1>/dev/null 2>&1; then
           phase_id=$(basename "$dir" | sed -E 's/^([0-9.]+)-.*/\1/')
           echo "Phase $phase_id has SUMMARY.md — checking ROADMAP.md..."
           if ! grep -q "Phase $phase_id.*✅" .quantis/ROADMAP.md; then
               echo "⚠️ Phase $phase_id completed but NOT marked ✅ in ROADMAP.md"
           fi
       fi
   done
   ```
3. **If mismatches found:**
   - Update ROADMAP.md to mark completed phases as `✅ Complete`
   - Report what was synced in the pause output:
     ```
     ⚠️ ROADMAP SYNC: Marked Phase {N} as ✅ Complete (was missing)
     ```
4. **If no mismatches:** Continue silently to Step 2

---
```
